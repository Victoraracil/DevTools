DevTools - Microherramientas

Elige una herramienta accesible y práctica para tu flujo de trabajo diario.

  • JSON a CSV

    Convierte objetos JSON a tablas CSV para copiar o descargar rápidamente.

  • CSV a JSON

    Convierte CSV con o sin cabecera en JSON legible y descargable.

  • PDF a Excel

    Extrae texto de PDF y genera archivo XLSX (texto plano).

  • Excel a PDF

    Convierte hojas de cálculo Excel a PDF con formato de tabla.

  • Generador de contraseñas

    Genera contraseñas seguras con opciones de longitud y símbolos.

  • Editor de PDF

    Carga, anota, dibuja y edita archivos PDF. Visualiza y descarga modificaciones.
